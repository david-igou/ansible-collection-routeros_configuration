david_igou.routeros_configuration.backup role
=============================================

Exports the running configuration as a text ``.rsc`` to the controller, and optionally takes an on-device binary backup.

See the :ansplugin:`full role reference <david_igou.routeros_configuration.backup#role>` for all parameters and defaults.
