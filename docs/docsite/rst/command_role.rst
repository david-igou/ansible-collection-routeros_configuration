david_igou.routeros_configuration.command role
==============================================

Escape hatch for arbitrary RouterOS API operations (``cmd`` / ``add`` / ``remove`` / ``update``). Explicitly not idempotent.

See the :ansplugin:`full role reference <david_igou.routeros_configuration.command#role>` for all parameters and defaults.
