david_igou.routeros_configuration.restore role
==============================================

Restores configuration: loads a binary backup (reboots the device) and/or imports a ``.rsc`` script with ``/import``. The reverse of ``backup``.

See the :ansplugin:`full role reference <david_igou.routeros_configuration.restore#role>` for all parameters and defaults.
