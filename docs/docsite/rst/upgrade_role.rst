david_igou.routeros_configuration.upgrade role
==============================================

Manages RouterOS package updates: sets the update channel, checks for updates, and optionally installs them (reboots). Optionally upgrades RouterBOARD firmware.

See the :ansplugin:`full role reference <david_igou.routeros_configuration.upgrade#role>` for all parameters and defaults.
