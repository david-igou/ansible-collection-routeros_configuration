david_igou.routeros_configuration.user_password role
====================================================

Sets or rotates passwords for existing ``/user`` accounts: the write-only field that ``configure`` cannot manage.

See the :ansplugin:`full role reference <david_igou.routeros_configuration.user_password#role>` for all parameters and defaults.
